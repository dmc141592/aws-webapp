Willkommen zur AWS Web-Applikation

Diese Website demonstriert die Verwendung verschiedener Amazon Web Services in einer modernen Cloud-Infrastruktur.

Die Anwendung nutzt folgende Services:

  • EC2 für das skalierbare Hosting
  • RDS für sichere Benutzerdaten
  • S3 für effiziente Dokumentenspeicherung
  • CloudFront für globale Performance
