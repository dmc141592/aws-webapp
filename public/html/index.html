Willkommen zur AWS Web-Applikation

Diese Website demonstriert die Verwendung verschiedener Amazon Web Services.

Die Anwendung verwendet folgende Cloud Services:

  • EC2 für Hosting
  • RDS für Benutzerdaten
  • S3 für Dokumente
  • CloudFront für schnelle Auslieferung

Projektbeschreibung

Diese Webanwendung enthält eine Themenseite sowie einen geschützten Downloadbereich. Dokumente können nur von angemeldeten Benutzern heruntergeladen werden.
