Das Projekt im Detail

Diese Web-Applikation demonstriert die Cloud-Native Entwicklung auf Amazon Web Services. Durch die Kombination verschiedener Managed Services erreichen wir hohe Skalierbarkeit und Sicherheit.

Projektziel

Ziel ist die Bereitstellung einer sicheren Plattform mit rollenbasiertem Zugriff. Dokumente und ZIP-Dateien sind physisch getrennt vom Webserver gespeichert und nur nach Authentifizierung abrufbar.

Verwendete AWS-Services

  • Amazon EC2: Rechenleistung für das Node.js Backend.
  • Amazon RDS: Strukturierte Speicherung von Nutzerdaten in einer MySQL-Instanz.
  • Amazon S3: Sicherer Objektspeicher für die PDF- und ZIP-Downloads.
  • Amazon CloudFront: Content Delivery Network (CDN) für minimale Ladezeiten.

System-Architektur

User → CloudFront → EC2 (Express.js) → RDS / S3

Die Architektur stellt sicher, dass der direkte Zugriff auf Ressourcen (S3) unterbunden wird und Anfragen immer über die Sicherheitsinstanzen laufen.

Technische Umsetzung

Das Backend nutzt Node.js und Express.js für das Routing. Die Verbindung zur Datenbank erfolgt über verschlüsselte Verbindungen direkt innerhalb der AWS VPC (Virtual Private Cloud).
