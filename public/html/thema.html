Thema: AWS Web-Applikation mit geschütztem Downloadbereich

Dieses Projekt zeigt die Entwicklung und Bereitstellung einer Web-Applikation auf Amazon Web Services. Die Anwendung kombiniert mehrere AWS-Dienste, um Hosting, Speicherung, Authentifizierung und Datenverwaltung umzusetzen.

Projektziel

Ziel des Projekts ist die Erstellung einer Website mit mehreren Inhaltsbereichen und einem geschützten Downloadbereich. Dokumente und ZIP-Dateien sollen nur von angemeldeten Benutzern heruntergeladen werden können.

Verwendete AWS-Services

  • Amazon EC2: Hosting der Web-Applikation
  • Amazon RDS: Speicherung von Benutzern, Rollen und Rechten
  • Amazon S3: Speicherung von Dokumenten und ZIP-Dateien
  • Amazon CloudFront: schnelle Auslieferung von Inhalten

Architektur

Die Anwendung folgt einer Architektur, bei der der Benutzer über CloudFront auf die Website zugreift. Die Web-Applikation läuft auf einer EC2-Instanz, greift auf Benutzerdaten in Amazon RDS zu und verwaltet Dateien in Amazon S3.

Ablauf:

User → CloudFront → EC2 → RDS / S3

Technische Umsetzung

Das Frontend wird mit HTML, CSS und JavaScript umgesetzt. Für das Backend wird Node.js verwendet. Zusätzlich kommt Express.js zum Einsatz, um Routing, Request-Verarbeitung und die Serverstruktur einfacher und übersichtlicher umzusetzen.

Die Anwendung wird später um eine MySQL-Datenbankverbindung, ein Login-System und einen geschützten Downloadprozess erweitert.
